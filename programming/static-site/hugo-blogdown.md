# Hugo/Blogdown

## Hugo Basics

[Quick Start | Hugo](https://gohugo.io/getting-started/quick-start/)

```bash
$ hugo new site mysite
$ cd mysite

# Serve local site (live reload)
$ hugo server

# (including drafts)
$ hugo server --buildDrafts

# (greedy loading mode)
$ hugo server --disableFastRender

# Build site
$ hugo
```
